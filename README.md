# Single Cell Tools

This repository contains a collection of tools for basic processing of single cell datasets.

## Tools Included
- compress_h5ad: Compress h5ad files to reduce file size.

## Installation:
Install from my Github repo using uv (or equivalent):

```bash
uv install git+https://github.com/austinv11/single-cell-tools.git
```

Or install as a package using uvx (or equivalent):

```bash
uvx install git+https://github.com/austinv11/single-cell-tools.git
```

## Usage:

```bash
compress_h5ad input_file.h5ad  # Inline compression
```
